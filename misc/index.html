ProfileMatic

ProfileMatic automates your phone settings (such as setting profile) based on rules defined by the user. Features:
  • Change of settings based on conditions such as time/weekday, location, WLAN
  • Changing of settings such as profile, availability, flight mode, power saving mode etc.
For questions and requests, please use ProfileMatic's support thread at TMO

Available for download from Nokia's Ovi Store.

Custom action tips

If you have more useful custom action tips to share, please post them to ProfileMatic's support thread at TMO.

Explanation and custom action QR code (use MeeScan to copy and paste)
Stand-by screen off: 
gconftool-2 -s --type boolean /system/osso/dsm/display/use_low_power_mode false
Stand-by screen on: 
gconftool-2 -s --type boolean /system/osso/dsm/display/use_low_power_mode true
Allow background connections off: 
gconftool-2 -s --type boolean /system/osso/connectivity/network_type/restricted_mode false
Allow background connections on: 
gconftool-2 -s --type boolean /system/osso/connectivity/network_type/restricted_mode true
