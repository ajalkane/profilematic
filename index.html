profilematic by ajalkane

Automatic profile changer for MeeGo Harmattan

ProfileMatic changes automatically profile according to user specified rules. At the moment you can change profile, and if it's the "ringing" profile also the volume, by specifying the the time when the profile is changed and the days when the rule is active. This is just a start, as I am also looking into adding for example location based rules for changing the profile. But before I go into that, I am intending to do a couple of release iterations on the time based rules depending on users' input. So if there's something you need or find lacking, let me know here. It can be suggestions on the UI to make it easier to understand (or fancier), or some extra functionality. Well, anything, but my top priority on the couple next releases is on the time based rules and the overall UX of the application.
