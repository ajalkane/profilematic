profilematic by ajalkane

Automatic profile changer for MeeGo Harmattan

ProfileMatic changes automatically profile (+ more) according to user specified rules.

Conditions for actions include:

  • Start/End time and day
  • Location by cell ids
  • WLAN
Actions to be done when conditions are fulfilled include:
  • Profile (and volume for ringing profile)
  • BlueTooth on/off
